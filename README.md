# Mandoo Simple

<p align="center">
  <img src="assets/mandoo-mascot.png" alt="Mandoo Simple pixel art mascot" width="180">
</p>

Mandoo Simple is a quiet warm-gray Typora theme for focused writing, technical notes, and PDF export.

It includes both light and dark variants, with tuned styles for code blocks, Mermaid diagrams, source mode, alerts, tables, footnotes, sidebars, popups, and print output.

## Themes

- `mandoo-simple.css` - light theme
- `mandoo-simple-dark.css` - dark theme

## Features

- Warm paper-like reading surface
- Wider writing area for technical notes
- Light and dark theme pair
- Dark-mode support for code blocks and Mermaid
- Polished alerts, task lists, footnotes, images, and tables
- Improved source mode colors
- Cleaner Typora sidebar, quick open, modal, dropdown, and popup colors
- PDF export rules for white paper, clean shadows, page breaks, and readable text

## Installation

1. Open Typora.
2. Go to `File` -> `Preferences` -> `Appearance`.
3. Click `Open Theme Folder`.
4. Copy these files into the theme folder:

```text
mandoo-simple.css
mandoo-simple-dark.css
```

5. Restart Typora.
6. Select `Mandoo Simple` or `Mandoo Simple Dark` from the theme menu.

## PDF Export Notes

The dark theme exports PDFs with a light print surface. Text, links, code blocks, and Mermaid labels are adjusted for white paper so the exported document stays readable.
